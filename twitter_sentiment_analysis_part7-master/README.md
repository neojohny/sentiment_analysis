```
pip install gensim
```
varying n-gram phrase detected corpus experimented with below models
- DBOW (Distributed Bag of Words)
- DMC (Distributed Memory Concatenated)
- DMM (Distributed Memory Mean)
- DBOW + DMC
- DBOW + DMM
